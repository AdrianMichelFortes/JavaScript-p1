# 🌐 Estudos em HTML & JavaScript

Durante meus estudos em desenvolvimento web, desenvolvi uma base sólida em conceitos de Front-End, unindo a estruturação de páginas com HTML5 e a interatividade lógica com JavaScript. Ao longo do aprendizado, pratiquei a criação de interfaces e a manipulação do DOM, aplicando conceitos que vão desde a lógica de programação até recursos avançados do JavaScript moderno, como a Programação Orientada a Objetos.

Este repositório reúne exercícios e projetos práticos desenvolvidos durante meu processo de aprendizagem, demonstrando minha evolução na estruturação e dinamismo de aplicações web.

---

# 🚀 Principais Conhecimentos Adquiridos

## 📌 Estruturação com HTML5

- ✔️ Semântica Web (tags apropriadas para acessibilidade e SEO)
- ✔️ Estruturação de textos, títulos, parágrafos e listas
- ✔️ Criação de tabelas e formulários de entrada de dados
- ✔️ Links de navegação e incorporação de mídias (imagens, vídeos)
- ✔️ Integração de scripts externos e folhas de estilo

---

## 📌 Lógica e Dinamismo com JavaScript

- ✔️ Tipos de dados, variáveis (`let`, `const`) e operadores
- ✔️ Estruturas condicionais (`if`, `else`, `switch`)
- ✔️ Estruturas de repetição (`for`, `while`, `forEach`)
- ✔️ Funções, Arrow Functions e escopo de variáveis
- ✔️ Manipulação do DOM (seleção e modificação de elementos HTML)
- ✔️ Tratamento de eventos (cliques, envios de formulários, digitação)

---

## 📌 Programação Orientada a Objetos (POO) em JavaScript

- ✔️ Criação de Classes e Construtores (`constructor`)
- ✔️ Instanciação de Objetos
- ✔️ **Herança** (utilização de `extends` e `super`)
- ✔️ Encapsulamento e Métodos de instância
- ✔️ Manipulação de objetos literais e JSON

---

## 📌 Estilização Básica com CSS3

- ✔️ Introdução à estilização e seletores básicos
- ✔️ Modificação de cores, fontes e alinhamentos simples
- ✔️ Aplicação de estilos voltada para o aprendizado prático de Front-End

---

# 💻 Projetos Desenvolvidos

Durante os estudos foram desenvolvidos sites de pequeno porte com o intuito de consolidar o aprendizado em Front-End, incluindo:

- 📌 Criação de páginas estáticas estruturadas de forma semântica
- 📌 Formulários interativos com validação de dados via JavaScript
- 📌 Projetos práticos aplicando herança e lógica de objetos em JS
- 📌 Exercícios de manipulação dinâmica do HTML através do DOM
- 📌 Pequenas interfaces estilizadas com foco em lógica de componentes

---

# 🎯 Objetivo

Este repositório tem como objetivo registrar minha evolução nos estudos de desenvolvimento web, consolidando os pilares do Front-End (HTML, CSS e JavaScript).

Os projetos e exercícios presentes aqui servem como base para a criação de aplicações web mais complexas, páginas dinâmicas e o futuro aprendizado de frameworks e bibliotecas modernas.
